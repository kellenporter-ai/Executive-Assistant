Gemini Executive Assistant — Setup

Let's get your assistant up and running. Follow the steps below for your operating system.

1
Install Python 3
  1. Go to python.org/downloads
  2. Click the big "Download Python 3.x.x" button
  3. Run the installer. On the first screen, look for the checkbox at the bottom:
Check "Add python.exe to PATH" — this is the #1 setup mistake. Without it, your terminal won't find Python. The checkbox is at the very bottom of the installer's first screen.
  1. Click "Install Now"
  2. When done, close and reopen any terminal/PowerShell windows
2
Install Node.js
  1. Go to nodejs.org
  2. Download the LTS version
  3. Run the installer — accept all defaults
3
Install Gemini CLI

Open PowerShell or Command Prompt and run:

npm install -g @google/gemini-cli
4
Launch the Assistant

Double-click start.bat in the Gemini Assistant folder.

Install Python 3

1
Install Python 3

Python may already be installed. Check first:

python3 --version

If that prints a version number, skip to Step 2. Otherwise, install for your distro:

sudo apt update && sudo apt install python3 python3-venv
sudo dnf install python3
sudo pacman -S python
2
Install Node.js
sudo apt install nodejs npm
sudo dnf install nodejs npm
sudo pacman -S nodejs npm
3
Install Gemini CLI

Open a terminal and run:

npm install -g @google/gemini-cli
4
Launch the Assistant

Open a terminal in the Gemini Assistant folder and run:

./start.sh

Verify Installation

Open a new terminal window and run each command. You should see version numbers, not errors.

python3 --version

Expected: Python 3.11.x or higher

node --version

Expected: v18.x or higher

gemini --version

Expected: a version number (any)

Windows users: If python3 is not found, try python --version instead. If neither works, re-run the Python installer and make sure "Add python.exe to PATH" is checked.
Having trouble? Ask your school's IT department to install Python 3 and Node.js on your computer.
