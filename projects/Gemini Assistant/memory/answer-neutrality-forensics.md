---
name: answer-neutrality-forensics
description: Dossier and status labels must remain neutral until forensic evidence links a suspect
type: feedback
---

Suspect dossiers should never use labels like "Primary Suspect" or include text hints that confirm guilt before the student performs the forensic match.
**Why:** "Answer leakage" in the UI ruins the investigative challenge.
**How to apply:** All potential database hits should be labeled neutrally (e.g., "Potential Match") and biographies should focus on professional background rather than crime-linking details.
